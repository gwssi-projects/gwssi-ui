# 长城软件基础UI组件及应用项目

基于elementUI 2.3.4

# 包含项目

``` bash
西安军民融合大数据中心 二期项目
```

# 关于启动项目

## Environment

`Node >= 6`

## Start

 - Clone or download this repository
 - Enter your local directory, and install dependencies:

``` bash
yarn
```

## Develop

``` bash
# serve with hot reload at localhost:8010
npm run dev
```

## Build

``` bash
# build for production with minification
npm run build
```
